Generating an href File

There are multiple ways to automatically generate an href file:

  • Generate an href file in a separate elaboration step by using the -genhref filename option.
  • Let the elaborator generate the files automatically when you create the simulation snapshot.
  • Let the software generate the href file internally. When using a single xrun command, you build all primary partitions along with the incremental partition, and this command automatically runs -genhref to resolve hierarchical references in either a primary or the incremental partition.
  • Generate a dump type-based href file along with instance-based href file using -genhref and -autohref options. This feature is covered under -gentypehref option.
