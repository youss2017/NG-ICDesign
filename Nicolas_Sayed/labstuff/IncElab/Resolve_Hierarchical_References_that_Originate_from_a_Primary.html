Resolve Hierarchical References that Originate from a Primary

MSIE supports hierarchical references to some kinds of objects that originate in a primary partition with the -genhref option. In addition to generating an href file, this option produces a primary_externs.v file for a primary if a design unit in that primary needs access to some other part of the design or if it needs access to some shared object that is in another partition.
